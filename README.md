# 🎯 কুইজ - বাংলা কুইজ ওয়েব অ্যাপ

একটি আধুনিক, পরিষ্কার এবং সম্পূর্ণ মোবাইল-ফ্রেন্ডলি বাংলা কুইজ ওয়েব অ্যাপ্লিকেশন।

## ✨ বৈশিষ্ট্যসমূহ

### 🔐 অ্যাকাউন্ট সিস্টেম
- **নিরাপদ লগইন/রেজিস্ট্রেশন**
  - ৪-२० অক্ষরের নাম
  - ঠিক ४ অঙ্কের পাসওয়ার্ড
- **স্বয়ংক্রিয় ১০ কয়েন** নতুন অ্যাকাউন্টে
- **স্থানীয় স্টোরেজে ডেটা সংরক্ষণ**

### 💰 কয়েন সিস্টেম
- প্রতি সঠিক উত্তর: +४ 🪙
- প্রতি ভুল উত্তর: -१ 🪙
- প্রবেশ ফি: ५ 🪙 (প্রথমবার)
- আবার খেলুন মোডে কোন খরচ নেই

### 🎮 কুইজ বৈশিষ্ট্য
- ৫ টি কুইজ সেট (খেলাধুলা বিষয়ক)
- প্রতি সেটে ५ টি প্রশ্ন
- **MCQ এবং টেক্সট-ভিত্তিক প্রশ্ন**
- **१३ সেকেন্ডের সময়সীমা** প্রতি প্রশ্নে
- স্বয়ংক্রিয় সময়োত্তীর্ণ স্কিপ
- বিস্তারিত ফলাফল পৃষ্ঠা

### 🎁 উপহার বাক্স
- **বিশেষ কোড সংগ্রহ করুন:**
  - `CADET` → ३० 🪙
  - `SCC` → २० 🪙
- প্রতিটি কোড শুধুমাত্র একবার ব্যবহারযোগ্য

### 👤 আমার অ্যাকাউন্ট
- প্রোফাইল তথ্য
- অ্যাকাউন্ট সৃষ্টির তারিখ
- মোট কুইজ খেলা
- নাম এবং পাসওয়ার্ড পরিবর্তন

### 📊 অগ্রগতি ট্র্যাকিং
- সামগ্রিক অগ্রগতি বার
- বিভাগ-ভিত্তিক অগ্রগতি
- প্রতি সেটের সম্পূর্ণতা স্ট্যাটাস

## 🚀 দ্রুত শুরু করুন

### প্রয়োজনীয়তা
- আধুনিক ওয়েব ব্রাউজার
- JavaScript সক্ষম
- স্থানীয় স্টোরেজ সমর্থন

### ইনস্টলেশন

1. **রেপোজিটরি ক্লোন করুন:**
```bash
git clone https://github.com/rathersakib24-gif/quiz-app.git
cd quiz-app
```

2. **ফাইলগুলি খুলুন:**
```bash
# সরাসরি index.html খুলুন আপনার ব্রাউজারে
```

3. **সার্ভার চালান (ঐচ্ছিক):**
```bash
python -m http.server 8000
# বা
npx http-server
```

তারপর `http://localhost:8000` এ যান।

## 📋 ব্যবহার

### অ্যাকাউন্ট তৈরি করুন
1. **অ্যাপ খুলুন**
2. **"অ্যাকাউন্ট তৈরি করুন" ক্লিক করুন**
3. **নাম এবং পাসওয়ার্ড লিখুন**
4. **অ্যাকাউন্ট তৈরি করুন বাটন ক্লিক করুন**
5. **স্বয়ংক্রিয়ভাবে १० 🪙 পাবেন**

### কুইজ খেলুন
1. **হোমপেজ থেকে একটি সেট নির্বাচন করুন**
2. **"খেলুন" বাটন ক্লিক করুন**
3. **কুইজ নিয়ম পড়ুন এবং স্বীকৃতি দিন**
4. **প্রতিটি প্রশ্নের উত্তর দিন**
5. **ফলাফল দেখুন এবং কয়েন অর্জন করুন**

### উপহার সংগ্রহ করুন
1. **সাইডবারে "উপহার বাক্স" খুলুন**
2. **একটি বৈধ কোড লিখুন (CADET বা SCC)**
3. **"সংগ্রহ করুন" বাটন ক্লিক করুন**
4. **তাত্ক্ষণিক কয়েন যোগ হবে**

## 💾 ডেটা স্টোরেজ

সমস্ত ডেটা **localStorage** এ সংরক্ষিত হয়:

```javascript
// ব্যবহারকারী ডেটা
{
  quizUsers: {
    [userName]: {
      name: string,
      password: string (4 digits),
      coins: number,
      createdAt: ISO string,
      totalQuizPlayed: number,
      usedGiftCodes: array,
      quizProgress: object
    }
  },
  currentUser: {
    // লগইনকৃত ব্যবহারকারীর ডেটা
  }
}
```
